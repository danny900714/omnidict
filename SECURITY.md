# Security Policy

## Supported Versions

Only the latest version is supported. Users of this project are encouraged to upgrade to the latest version.

| Version        | Supported          |
|----------------|--------------------|
| latest version | :white_check_mark: |
| other versions | :x:                |

## Reporting a Vulnerability

**Do not submit an issue or pull request**: this might reveal the vulnerability.

Instead, you should use the form
to [privately report a vulnerability to us via GitHub](https://github.com/danny900714/omnidict/security/advisories/new).

We will deal with the vulnerability privately and submit a patch as soon as possible.
